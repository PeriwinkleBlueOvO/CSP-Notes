# CSP 备考笔记：从 C++ 基础到认证实战

> 一份面向 CSP-J/S / CCF CSP 认证备考的算法学习笔记。内容来自个人备考过程，重点不是堆算法名词，而是把考场上真正容易用到的 C++ 基础、STL、模拟、枚举、贪心、二分、前缀和、差分、DP、图论基础等知识点整理成可复习、可迁移、可实战的材料。

## 项目定位

这个仓库记录的是我备考 CSP 过程中的系统笔记、题型总结和做题复盘。

适合：

- 正在准备 CSP 认证，但基础还不够稳的同学；
- 想把 A / B 题做稳，C / D题尽量拆出来的同学；
- 学过一些算法，但考场上容易忘模板、乱边界、写不出来的同学；
- 希望按专题复习 C++ 算法基础的同学。

不适合：

- 想直接冲高阶竞赛算法的同学；
- 想找完整题库题解合集的同学；
- 想跳过基础，直接背代码模板的同学。

## 备考结果

本人在最近一次 CSP 认证中取得 **260 分**，大致位于 **全国前 6%**。这份笔记主要服务于 CSP 考场实战，尤其关注：

- A 题快速稳定 AC；
- B 题减少模拟与边界错误；
- C 题提高拆题与部分分能力；
- D / E 题在能力范围内争取可拿分部分。

## 推荐阅读顺序

1. C++ 基础与输入输出
2. STL 常用容器与函数
3. 模拟题常见写法
4. 前缀和与差分
5. 二分答案
6. 贪心与排序
7. 动态规划基础
8. 位运算
9. 高精度
10. 图论基础

## 笔记特点

- 先讲原理，再讲题型；
- 重点记录“什么时候用”和“为什么这样写”；
- 保留常见错误与调试经验；
- 每道题复盘固定记录：未一次 AC 原因、知识点类型、题解、关键点、注意事项；
- 尽量用 CSP 考场视角组织内容，而不是单纯按照算法书目录堆知识点。

## 题目来源说明

笔记中可能会引用洛谷、曙梦 OJ / 水木 OJ 等平台题目。题目版权归原平台所有，本仓库只整理个人学习笔记、解题思路与复盘，不搬运平台题面全文。

为了避免版权问题，仓库中：

- 不整段复制题面；
- 只写题号、题目名称、来源链接；
- 重点记录自己的思路、错误、代码与复盘；

## 如何使用这份笔记

建议不要直接从头到尾“看完”。更有效的方式是：

1. 先选一个专题，例如二分、贪心或 DP；
2. 看对应知识点和模板；
3. 做一题；
4. WA / TLE / CE 后回来看错点；
5. AC 后写复盘；
6. 再进入下一题。
